# Frontend Agent

## Goal
Build a React + Vite + TypeScript customer platform for discovery, authentication, checkout, payments, order tracking, and admin access.

## Tasks
- Configure Vite, TailwindCSS, shadcn-style components, React Query, React Hook Form, Zod, and Framer Motion.
- Implement routing and API client.
- Build pages for landing, services, checkout, account, auth, and admin.
- Connect checkout to payment endpoints before order creation.

## Files to create
- `frontend/src/main.tsx`
- `frontend/src/App.tsx`
- `frontend/src/lib/api.ts`
- `frontend/src/pages/*.tsx`
- `frontend/src/components/*.tsx`

## Completion checklist
- [ ] `npm run dev` starts Vite.
- [ ] Customer flow works end to end.
- [ ] Admin dashboard reads backend data.
- [ ] Forms validate with Zod and React Hook Form.
