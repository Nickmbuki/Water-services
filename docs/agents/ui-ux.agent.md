# UI UX Agent

## Goal
Create a professional large-company water services interface for Nairobi and Kiambu customers.

## Tasks
- Build a full-width tanker truck hero.
- Add service imagery for tanker delivery, borehole drilling, plumbing, and water systems.
- Keep phone numbers visible in contact areas and footer.
- Use responsive layouts, clear CTAs, and refined motion.
